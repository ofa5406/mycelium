MYCELIUM

MYCELIUM is an experimental app exploring whether human cognition is becoming interconnected through real-time conversation and micro-interactions.

---

OVERVIEW

The app combines:

- MYCI → a conversational interface that helps users think and decide
- Probe System → lightweight, dynamic micro-interactions that capture cognitive signals

Together, they explore collective intelligence emerging from human input.

---

SETUP

1. Clone the repository

git clone https://github.com/your-username/mycelium.git
cd mycelium

2. Install dependencies

npm install

3. Create .env file

Create a .env file in the root directory and add:

CLAUDE_API_KEY=your_api_key_here

4. Run the app

npm run dev

---

REQUIREMENTS

- Claude API key (Anthropic)
- Node.js (v18+ recommended)

---

NOTES

- Claude powers the MYCI conversational layer
- Probe system generates dynamic micro-interactions
- System is simple on the surface, complex underneath
